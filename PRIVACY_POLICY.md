# Privacy Policy — ALT-DEV TOOLS

**Last Updated:** January 25, 2025

## Overview

ALT-DEV TOOLS ("the Extension") is a browser-based developer tools sidebar extension for Google Chrome. This privacy policy explains how the Extension handles user data.

## Data Collection

**ALT-DEV TOOLS does not collect, store, or transmit any personal data.**

The Extension operates entirely within your browser. All information displayed in the developer tools panels — including DOM trees, console output, network requests, cookies, and storage data — is processed locally and is never sent to any external server.

## What Information the Extension Accesses

To provide its developer tools functionality, the Extension accesses the following information **locally within your browser**:

- **DOM Structure**: The HTML structure of the web page you are inspecting, displayed in the Elements panel.
- **Console Output**: JavaScript console messages (log, warn, error, etc.) generated by the web page.
- **Network Requests**: XHR and fetch requests made by the web page, including headers and response bodies.
- **Cookies**: Cookies associated with the current web page domain, displayed in the Application panel.
- **Local/Session Storage**: Key-value pairs stored in the browser's localStorage and sessionStorage for the current page.
- **Page Source Files**: JavaScript, CSS, and HTML source files loaded by the current web page.

**All of this information is only displayed in the Extension's sidebar panel and is never transmitted, logged, or stored externally.**

## Data Storage

The Extension uses `chrome.storage.local` solely to persist user preferences (such as panel layout settings and filter configurations). No personal data, browsing data, or page content is stored.

## Third-Party Services

ALT-DEV TOOLS does **not** use any third-party services, analytics, tracking, or advertising. No data is shared with any third parties.

## Remote Code

ALT-DEV TOOLS does **not** load or execute any remote code. All scripts are bundled within the extension package. The Extension uses `chrome.scripting.executeScript()` only to inject pre-packaged content scripts that are included in the Extension's installation package.

## Permissions Used

| Permission | Purpose |
|------------|---------|
| `activeTab` | Access the current tab for DOM inspection |
| `cookies` | Display cookies in the Application panel |
| `host_permissions` | Inject content scripts for developer tools functionality |
| `scripting` | Fallback content script injection |
| `sidePanel` | Display the developer tools UI as a sidebar panel |
| `storage` | Save user preferences locally |
| `tabs` | Track active tab for panel synchronization |

## Children's Privacy

ALT-DEV TOOLS is a developer tool not intended for use by children under 13 years of age. We do not knowingly collect information from children.

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date above.
